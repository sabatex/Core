# Sabatex Core Libraries

[![MIT License](https://img.shields.io/badge/license-MIT-red.svg)](https://github.com/sabatex/Core/blob/master/LICENSE.TXT)

Набір базових бібліотек Sabatex для роботи з обміном об'єктами, даними та інтеграції з різними системами.

## 📚 Бібліотеки

| Пакет | Версія | Опис |
|-------|--------|------|
| **Sabatex.Core** | [![NuGet](https://img.shields.io/nuget/v/Sabatex.Core.svg)](https://www.nuget.org/packages/Sabatex.Core) | Базові утиліти та розширення |
| **Sabatex.ObjectExchange.Core** | [![NuGet](https://img.shields.io/nuget/v/Sabatex.ObjectExchange.Core.svg)](https://www.nuget.org/packages/Sabatex.ObjectExchange.Core) | Ядро системи обміну об'єктами |
| **Sabatex.ObjectExchange.Models** | [![NuGet](https://img.shields.io/nuget/v/Sabatex.ObjectExchange.Models.svg)](https://www.nuget.org/packages/Sabatex.ObjectExchange.Models) | Моделі даних для API обміну |
| **Sabatex.ObjectExchange.ClientDataAdapter** | [![NuGet](https://img.shields.io/nuget/v/Sabatex.ObjectExchange.ClientDataAdapter.svg)](https://www.nuget.org/packages/Sabatex.ObjectExchange.ClientDataAdapter) | Клієнтський адаптер даних (EF Core) |

## 🚀 Швидкий старт

```bash
# Встановлення базової бібліотеки
dotnet add package Sabatex.Core

# Встановлення клієнтського адаптера
dotnet add package Sabatex.ObjectExchange.ClientDataAdapter
```
